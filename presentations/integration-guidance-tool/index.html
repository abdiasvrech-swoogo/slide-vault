The "Act + Ask + Expand" Pattern

ACT — Targeted Discovery

Infer specific scope and pull integration guide for that domain only.

2

ASK — Data-Driven Prompting

Suggest next domains (e.g., payments) based on the event data discovered.

3

EXPAND — Controlled Scaling

Add documents one-by-one only as the user confirms. Keeps context lean.
